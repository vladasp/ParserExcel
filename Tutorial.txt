Приложение ParserExcelData предназначено для считывания и отображения данных из файла Excel.
Исходными данными для работы приложения является файл с данными в Excel.
В левом верхнем углу запущенного приложения находится кнопка отвечающая за предоставление пути к файлу. После выбора появится строка с полным путем и именем этого файла.
В интерфейсе присутствует индикатор для отображения процесса парсинга данных. 
Программа считывает данные с первого листа, левая крайняя ячейка является первой в таблице, а название колонок в верхнем ряду (данные упрощения были приняты в для ускорения процесса разработки и могут быть вынесены как настраеваемые параметры).

Рекомендация - поиск файла для парсинга начинаеется с папки в которой находится программа. Поэтому файл Excel необходимо поместить в папку с программой для ускорения поиска файла.

Данное приложение было выполнено средствами VisualStudio 2015. Для сборки, рефакторинга и дебага приложения необходимо использовать VS 2015.
